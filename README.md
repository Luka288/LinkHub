# LinkHub

A link management app similar to Linktree.

## Tech Stack

**Frontend:** Angular, TypeScript  
**Backend:** Node.js, Express, TypeScript  
**Database:** PostgreSQL (Neon)

## Features

- Register / Login with JWT auth
- Add, edit, delete links
- Toggle links on/off
- Public profile page
- Click tracking

## Projects

- `/frontend` - Angular app
- `/backend` - REST API
